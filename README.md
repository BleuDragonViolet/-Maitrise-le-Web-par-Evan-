## 🚀 Déploiement

Tu peux déployer ce site en quelques clics :

1. **Fork** ce repo
2. Clone-le :  
```bash
git clone https://github.com/tonpseudo/mon-site-web.git
````

3. Ouvre `index.html` avec Live Server (ou dans ton navigateur)
4. Publie-le depuis GitHub Pages :

   * Settings → Pages → Source : `main` / root
