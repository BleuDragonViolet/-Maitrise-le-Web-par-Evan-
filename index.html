📌 Introduction

Créer un site web peut sembler intimidant, mais avec les bons outils et un peu de pratique, c'est accessible à tous. Ce guide va t'accompagner étape par étape.

🌐 Qu'est-ce qu'un site web ?

Un site web est une collection de pages reliées entre elles, contenant du texte, des images, des vidéos, etc. Il est composé de 3 langages principaux :

  • HTML - Structure
  • CSS - Style
  • JavaScript - Interactivité

🧱 Les langages du web

HTML (HyperText Markup Language)

<h1>Bienvenue</h1>
<p>Ceci est un paragraphe.</p>

CSS (Cascading Style Sheets)

body {
  background-color: #121212;
  color: #fff;
}

JavaScript

function direBonjour() {
  alert("Bonjour !");
}

🚀 Mise en ligne avec GitHub Pages

  1. Crée un compte GitHub
  2. Crée un nouveau dépôt
  3. Upload tes fichiers
  4. Va dans Settings > Pages
  5. Sélectionne la branche main et dossier /
